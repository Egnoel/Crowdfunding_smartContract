## Installation and Setup

### Prerequisites

- forge
- Yarn

### Steps to Install

1. Clone the repository:

   ```bash
   git clone https://github.com/Egnoel/Crowdfunding_smartContract
   ```

2. Navigate to the project directory:

   ```bash
   cd Crowdfunding_smartContract
   ```

3. Install dependencies:

   ```bash
   yarn install
   ```
